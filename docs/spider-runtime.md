# Spider runtime image

`build/spider-rt.img` contains:

```text
/sbin/spider-rs
/etc/spider/default.target
/etc/spider/basic.target
/etc/spider/emergency.target
/manifest
```

The image is generated by `make spider-rt-image` using `tools/mk-runtime-image` and staged into the ISO as a Limine module with `mirage.module=spider-rt`.
